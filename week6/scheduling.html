Scheduling Notes

Scheduling Criteria

User Oriented

  • Turnaround time: interval between submission and completion; batch job measurement
  • Response time: time from submission of request until response begins to be received (key press to character displayed on screen)
  • Deadlines
  • Predictability: the same job runs with similar turnaround and response times regardless of load on system

System-Oriented

  • Throughput: number of processes completed per unit of time
  • Processor utilization: percentage of time that the processor is busy (think about IO/CPU bound processes)
  • Fairness: no starvation; processes should be treated roughly the same
  • Priorities
  • Balancing resources: there are other resources than just CPU time

Scheduling Algorithms

List of scheduling algorithms you are expected to know:

  • First Come First Served
  • Round Robin
  • Shortest Time to Completion First (with preemption; how do you know the time to completion?)
  • Exponential Queue/Multi-level Feedback Queues (what about starvation?)
